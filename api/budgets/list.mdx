---
title: "List Budgets"
api: "GET /budget"
description: "Get all budgets for an account"
---

Retrieves all budgets associated with a specific account, ordered by creation date (newest first).

## Query Parameters

<ParamField query="account_id" type="string" required>
  The UUID of the account to retrieve budgets for
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="array">
  Array of budget objects
  
  <Expandable title="Budget object">
    <ResponseField name="id" type="string">
      The unique UUID identifier for the budget
    </ResponseField>
    
    <ResponseField name="account_id" type="string">
      The UUID of the account this budget belongs to
    </ResponseField>
    
    <ResponseField name="category_id" type="string">
      The UUID of the category this budget applies to
    </ResponseField>
    
    <ResponseField name="amount" type="number">
      The budget amount in cents
    </ResponseField>
    
    <ResponseField name="period" type="string">
      The budget period: `weekly`, `monthly`, or `yearly`
    </ResponseField>
    
    <ResponseField name="alert_threshold" type="number">
      The percentage threshold for spending alerts (50-100)
    </ResponseField>
    
    <ResponseField name="created_at" type="string">
      ISO 8601 timestamp when the budget was created
    </ResponseField>
    
    <ResponseField name="updated_at" type="string">
      ISO 8601 timestamp when the budget was last updated
    </ResponseField>
  </Expandable>
</ResponseField>

```bash cURL

  -H "Authorization: Bearer YOUR_TOKEN"
```

## Error Codes

- `400` - Missing or invalid `account_id` parameter
- `401` - Missing or invalid authentication token
- `500` - Internal server error
